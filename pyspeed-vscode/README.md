# PySpeed

PySpeed is a lightweight Python performance profiler designed for developers who want **real-time insights while writing code**.

It displays **execution time and memory usage directly next to Python function definitions inside VS Code**.

Example:


## Features

- Real-time Python function profiling
- Execution time tracking
- Memory usage monitoring
- Inline performance display inside the editor
- Lightweight and fast

## Why PySpeed?

Traditional profilers require running the program after writing code.  
PySpeed brings **performance awareness directly into the coding workflow**.

## Planned Features

- Live profiling during execution
- Flame graph visualization
- Function call tracing
- Performance regression detection

## Usage

1. Install the extension
2. Open a Python file
3. Start profiling
4. View performance metrics next to function definitions
